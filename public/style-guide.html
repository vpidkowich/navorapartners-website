19. Responsive Breakpoints

CSS custom properties can't be used in media queries, so these are documented values.

Breakpoint Trigger Key Changes
1200px Tablet Spacing scale restores to 1x. Hamburger menu activates. Section padding reduces.
1200px + portrait Tablet Portrait Hero image hidden. Content centers. Accomplishments stats reflow to 3-column.
1080px Ring Diagram Ring diagram scales down slightly.
768px Mobile Single-column layouts. Cards stack. Buttons go full-width. CTA moves to hamburger.
600px Ring Diagram Mobile Ring diagram replaced by horizontal tab bar + content panel.
480px Small Mobile Further font size reductions. No layout changes.
